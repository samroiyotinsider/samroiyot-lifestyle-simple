# Sam Roi Yot Lifestyle - Properties & Concierge

A simple property listing website built with React, Express, and PostgreSQL.

## Features

- Public property listing page
- Admin panel for managing properties
- User authentication
- Responsive design

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm start
```

## Environment Variables

- `DATABASE_URL` - PostgreSQL connection string
- `JWT_SECRET` - Secret key for JWT tokens
- `PORT` - Server port (default: 3001)

## Deployment

This project is configured for Railway deployment. Simply push to GitHub and Railway will automatically build and deploy.
